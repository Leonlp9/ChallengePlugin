You need a Minecraft Paper Server. If you don't have one, you can download it from here

When you know about setting up a server, you can skip this part. Here's how to set up a server:

  1. Download the server from the link above
  2. Run the server once to generate the files. For Windows, you can run the start.bat file and for Linux, you can run the start.sh file
  3. Open the eula.txt file and change eula=false to eula=true
  4. Run the server again
  5. Next, you can install the plugin by following the steps below

Here's how to install the plugin:

  1. Download the plugin from Spigot
  2. Drag the plugin into your server's plugins folder
  3. Restart your server
  4. Enjoy!
